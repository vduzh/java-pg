## Installation

### Docker

* Install docker
* Check: docker ps

### Database (postgres)

* https://hub.docker.com/_/postgres
* docker run --name postgres-pg -e POSTGRES_PASSWORD=postgres -p 5433:5432 -d postgres
* Check if the postgres is available: docker ps
